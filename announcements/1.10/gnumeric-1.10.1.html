Gnumeric 1.10.1 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.10.1.

This is a bugfix release. There is nothing too exciting on the list.

  • Andreas
    • Fix storing of error-literals in ODF. [Bug 610175].
    • Don't export filter combos to ODF. [Bug 610399].
    • Apply filter on ODF import. [Bug 610399].
    • Fix display of autocomplete suggestions. [Bug 610537].
    • Fix formatting of multirow cells in LaTeX export. [Bug 611633].
    • Protect exported LaTeX files from babel shorthands.
  • Jean
    • Fix a crash in excelplugin. [Bug 610012].
    • Fix cursor appearance. [Bug 611076].
  • Morten
    • Turn off -Wstrict-prototypes for the stable series.
    • Fix ODF read problem. [Bug 610178].
    • Fix rendering problem with iterated evaluation.
    • Fix solver undo problem. [Bug 611406].
    • Fix graph label problem. [Bug 611362].
    • Fix GLPK solver issue. [Bug 611407].
    • Fix ssindex crash. [Bug 611842].
    • Fix ODF export of intersection. [Bug 611824].
    • Fix cell comment critical.
