Gnumeric 1.10.14 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.10.14. This release is best used with the concurrently released Goffice 0.8.14 and Libgsf 1.14.20.

Noteworthy news in this release:

  • XLSX import has been greatly improved for graphs.
  • ODF import and export of validation has been improved.
  • Gnumeric can use GSettings if Goffice has been compiled for it.
  • Print-Preview sort-of works on Win32.
