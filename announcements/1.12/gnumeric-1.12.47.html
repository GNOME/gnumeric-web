Gnumeric 1.12.47 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.47. This is mainly a bugfix release, but also features a few improvements to ssconvert, our command line spreadsheet converter.

Details of changes by contributor:

  • Andreas
    • Fix chart and image name roundtrip through ODF. [Bug 477].
    • Fix print flag roundtrip for sheet objects through ODF. [Bug 486].
  • Jean
    • Fix crash when the embedded spreadsheet is invalid. [Bug 481].
  • Morten
    • Fix dialog size problem.
    • Add "--set CELL=CONTENTS" option to ssconvert.
    • Improve tests.
    • Improve speed on exit with lots of conditional formatting.
    • Improve error message [Bug 472].
    • Impose ordering of ssconvert --export-graphs.
    • Doc fixes.
    • Fix CELL crash. [Bug 479].
    • Handle export options for ssconvert --export-graphs.
    • Fix deps crash. [Bug 480].
    • Fix chart name roundtrip through XLSX. [Bug 478].
    • Fix sheet object print flag roundtrip through XLSX. [Bug 486].

goffice 0.10.47 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.47. This is a bugfix release.

Details of changes by contributor:

  • Jean
    • Fix crash if a XY plot series has data labels but no valid data. [Bug 426].
    • Fix criticals related to data labels. [Bug 428].
    • Fix a drawing error in contour plots. [Bug 458].
    • Ensure that all required dims have valid data in a valid series. [Bug 466].
    • Fix legend order in bar plots.
    • Test plot series validity after a data change. [Bug 468] .
    • Fix crossing axis position. [Bug 45].
    • Fix component duplication. [Bug 483].
  • Morten
    • Fix library namespace issue.
    • Implement auto-by-extension for image files.
