Gnumeric 1.12.40 aka "Introspective" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.40. This version requires the concurrently released Goffice 0.10.40. The recently-released Libgsf 1.14.43 is recommended.

Noteworthy news in this release:

  • Major upgrades to introspection. It is now feasible to write a python script that reads files, applies changes, and writes them out again.
  • Catch files that change underneath Gnumeric, so we avoid overwriting them. (Requires updated Libgsf.)
  • Add GUI for comparing two sheets, presumably an old and a new version of the same sheet.
