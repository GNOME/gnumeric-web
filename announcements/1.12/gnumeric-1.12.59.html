Gnumeric 1.12.59 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.59. Note that we skipped 1.12.58 for technical reasons.

Noteworthy news in this release:

  • Problems with zoomed views fixed

Details of changes by contributor:

  • b.s.
    • Fix long-double version of FLT.NEXTAFTER.
    • cover denormals and excessive digits for TRUNC, ROUNDDOWN, ROUNDUP and ROUND. [Bug 770].
  • Morten
    • Fix xlsx import problem with broken file. [Bug 763].
    • Fix ods round trip of iteration tolerance. [Bug 765].
    • Fix issues with long doubles in dialogs.
    • Leak fixes.
    • Fix ssconvert --export-range for pdf. [Bug 783].
    • Fix invisible cursor problem. [Bug 712].
    • Fix zoom problem. [Bug 707].
    • Fix further zoom problem. [Bug 794].
