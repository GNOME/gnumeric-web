Gnumeric 1.12.34 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.34.

Noteworthy news in this release:

  • Hyperlinks fixes
  • xlsx/ods import fixes.

Details of changes by contributor:

  • Andreas
    • Fix hyperlinks import from ODF. [Bug 779635].
  • Jean
    • Fix xlsx export of chart series names. [Bug 778618].
    • Always export a transparent fill style in charts. [Bug 779160].
  • Morten
    • Doc fixes.
    • Fix xlsx import with missing header/footer. [Bug 778292].
    • Minor xlsx import improvements with alias colors.
    • Test suite improvements.
    • Improve Excel handling of hyperlinks.
    • Hyperlinks improvements. [Bug 706671].
    • Plug leaks.
    • Fix issues with sheet-local names.
    • ssdiff improvements with hyperlinks, input messages, conditional formats, and validations.
    • Make exit code from ssdiff follow diff's.
    • Fix ssdiff with inter-sheet references.
    • Fix style border colour sharing.
