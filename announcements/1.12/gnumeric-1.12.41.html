Gnumeric 1.12.41 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.41. This is a bugfix release.

Details of changes by contributor:

  • Morten
    • Fix problem with object pasting.
    • Fix undo problem for cell comments. [Bug 732653].
    • Exit with non-zero with we fail to load file on command line.
    • Fix undo crash. [Bug 795861].
    • Introspection fixes.
    • Don't offer more than 15 significant digits by default. [Bug 754391].
