Gnumeric 1.12.36 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.36.

Noteworthy news in this release:

  • ODF/XLSX import/export improvements with contour graphs.
  • Fix some GUI issues caused by Gtk+ "updates".

Details of changes by contributor:

  • Andreas
    • Improve contour plot export/import to/from ODF. [Bug 788728].
    • Improve contour plot import of Excel generated ODF. [Bug 788801].
    • Improve surface plot export to ODF. [Bug 788447].
    • Fix export/import of CONCATENATE to and from ODF.
    • Fix export/import of minor axis divisors to and from ODF.
    • Fix import of affice trendline from ODF. [Bug 789538].
  • Jean
    • Fix background of multiple selection. [Bug 789411].
    • Workaround for gtk scroll bar bug. [Bug 789412].
  • Morten
    • Convert all xpm files to png.
    • Avoid to-pixdata options for resources. It's going away.
    • Minor CORREL improvement.
    • Fix error on xlsx files with bogus Print_Area.
    • xlsx import improvements for contour graphs.
    • xlsx export improvements for contour graphs.
    • Fix xlsx export of trend line equation.
    • Fix xlsx import and export of trend line affine flag.
    • Avoid critical on exporting unnamed xlsx trend line.
    • Fix VDB for partial periods. [Bug 790299].

goffice 0.10.36 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.10.36. This is a bugfix release.

Details of changes by contributor:

  • Jean
    • Fix engineering format. [Bug 785669].
    • Fix crash in 3D plots when X or Y values are missing. [Bug 788437].
    • Don't use invalid styles in legend from contour or surface plots imported from Excel. [Bug 788861].
    • Fix documentation build with recent gtk-doc. [Bug 788710].
