Gnumeric 1.12.53 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.53.

Noteworthy news in this release: a crash related to re-using dialogs affecting newer Gtk+ versions was fixed.

Details of changes by contributor:

  • Morten
    • Fix xlsx import of T.INV.
    • Fix inter-process array paste problem. [Bug 634].
    • Fix problem with xlsx number format 14.
    • Fix problems with complex number rendering. [Bug 638].
    • Fix near-denormal parsing case. [Bug 656].
    • Fix mildly crazy ROUND edge cases. [Bug 661].
    • Fix dialog reuse problem. [Bug 657].
    • Fix sc import problem. [Bug 663].
    • Work a little harder at avoiding -0. [Bug 652].
    • Fix html import problem. [Bug 669].
    • Fix reading localc extension for engineering format. [Bug 659].
    • Fix strict-conformance ods problem with E-notation. [Bug 672].
    • Improve test suite.
