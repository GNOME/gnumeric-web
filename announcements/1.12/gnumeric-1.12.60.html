Gnumeric 1.12.60 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.60.

Noteworthy news in this release:

  • Improve gui experience on high-dpi display (where "high" really just means significantly different from 90, so not a high bar)
  • A year's worth of accumulated bug fixes.

Details of changes by contributor:

  • Morten
    • Fix problem with whole-column references in ods. [Bug 797].
    • ODS Export and import of marker fill and outline opacity. [Bug 796].
    • ODS Export and import of graph line stroke opacity. [Bug 796].
    • Fix XLSX import for files lacking cell addresses. [Bug 804].
    • GUI improvement when saving file loaded from cvs. [Bug 817].
    • Improve LOG with base argument.
    • Doc system improvements. [Bug 823].
    • Fix GEOMEAN problem. [Bug 829].
    • Fix dialog accel collision. [Bug 830].
    • Introspection improvements.
    • Named expression fixes. [Bug 833].
    • Speed up large-range copies. [Bug 836].
    • Add missing undo/redo for sheet display options.
    • Style the progress bar we use for menu tooltips.
    • Clean out old references to mailing list and irc.
    • Documentation updates.
    • Print using pango context from GtkPrintContext for constistency between gui and non-gui.
    • GUI files relating to cell sizes.
    • Fix rare undo problem.
    • Experiment with AI translation (+human editing) for the da.po files.
