Gnumeric 1.12.49 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.49. This release requires the concurrently released Goffice 0.10.49.

Noteworthy news in this release:

  • Internal handling of conditional formats have changed, greatly improving the performance for large-area formats.
  • We have had to disable the "cs" translation of the documentation due to a bug in intltool and/or libxml2. It's not the cs translation team's fault, they are just the victim here.
  • Styling improvements should make it possible to implement dark theming, but no-one has actually done so yet.
