Gnumeric 1.12.54 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.12.54. This is a bugfix release.

Details of changes by contributor:

  • John Denker
    • Improve whitespace handling in html import. [Bug 671].
  • Morten
    • Improve various desktop environments' icon discovery. [Bug 677].
    • New function ENCODEURL. [Bug 658].
    • New libpython configuration. [Bug 680].
    • Fix problem relating to sheet remove and conditional format.
    • Fix inconsistent rounding of half-way cases. [Bug 638].
    • Fix xlsx leaks.
    • Cell comment speed-ups. [Bug 692].
