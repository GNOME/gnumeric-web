Gnumeric 1.11.1 aka "Lots of Ones" is now available.

The Gnumeric Team is pleased to announce the availability of Gnumeric version 1.11.1. This release requires the concurrently released goffice 0.9.1

The release fixes build problems caused by a missing file in version 1.11.0's tarball.

  • Andreas
    • Show range selection cursors even while selecting new regions and on other sheets.
  • Jean
    • Fixed expression range cursor behavior. [Bug 664852].
  • Morten
    • Build fixes relating to embedded ui files [Bug 664828].

goffice 0.9.1 aka "TBD" is now available.

The Gnumeric Team is pleased to announce the availability of goffice version 0.9.1.
